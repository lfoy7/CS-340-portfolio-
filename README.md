# Grazioso Salvare Dashboard Portfolio
Leonard Foy 

This project includes my Project Two dashboard code and CRUD Python module. The dashboard connects to MongoDB and uses a table, filters, chart, and map to help Grazioso Salvare identify dogs for rescue training.

I made the program maintainable by separating the database code from the dashboard code. The CRUD module handles MongoDB operations, while the dashboard handles the user interface. This made the project easier to read, update, and reuse.

I approached the project by breaking the client’s needs into smaller parts: database connection, data table, filters, chart, and map. This helped me build and test one piece at a time.

Computer scientists solve problems by using technology and data. This project helps a company make faster decisions by turning animal shelter records into useful, searchable information.
